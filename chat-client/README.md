# chat-client

本项目是研究与开发实践课程大作业-在线聊天室的前端部分

借用了阿童木的前端界面部分，调整了接口，以对应我们的springBoot后端。

另外增加了一些功能需求以满足课程要求。

原项目链接：[genal-chat: 🚀阿童木聊天室 ](https://github.com/genaller/genal-chat/tree/master)

## dependcy

node v10.16.3
